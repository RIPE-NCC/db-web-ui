Database API Keys management

Maintainer API Keys allow you to script updates to the {{ environment }} database without needing to login to RIPE NCC Access. Read the documentation to find out how.
You can create as many different keys as you'd like. We suggest that you create a separate key for each application that you'd like to provide with access.
That way, if you believe the key has been compromised, you can revoke access to that application without affecting other applications that you have provided with access.
The keys you create on this page can only be used to make updates in the {{ environment }} database and not with any other RIPE NCC service.

My Resources API Keys allow you to automate access to all internal registry information about your organisation's resources, including ASNs, IPv4 and IPv6 allocations, PI assignments, etc.

IP Analyser API Keys also allow you to automate access to information about your organisation's assignments, available free space and invalid assignments.
